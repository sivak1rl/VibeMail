# Contributors

This project tracks contributors from git history.

## Core Contributors

- Rich Sivak <rich@richsivak.com>

## How To Be Listed

- Use your preferred name and email in git commits (`git config user.name` / `git config user.email`).
- Open a PR or commit with meaningful project changes.
- Contributors are periodically synced from:

```bash
git shortlog -sne --all
```
